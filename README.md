Fitness & Nutrition Coach Chatbot

Project Description:
FitCoach AI is your personal Fitness & Nutrition Coach chatbot built with Streamlit and LangGraph + Gemini 2.5 Pro.
Get customized workout plans, diet advice, supplement guidance, and recovery tips all in one interactive AI assistant.

Features:
>> Personalized workout programs for muscle gain, fat loss, and strength building
>> Custom meal plans & macro tracking
>> Advice on supplements & recovery
>> Interactive chat interface with dark theme UI
>> Full session memory for context-aware coaching

Tech Stack
Frontend: Streamlit (Python)
Backend: LangChain, Gemini 2.5 Pro
UI: Dark theme with CSS customization
Chat: Session-based conversational memory


Setup Instructions
>>>>>>>>>>>>>>>>>>>>....................>>>>>>>>>>>>>>>>>>>>.......................>>>>>>>>>>>>>>>>>>>>>>

step(1) Clone the repository:
git clone https://github.com/dhirajdhardip/Ai_chatbot.git
cd Ai_chatbot

step(2) Create and activate a virtual environment:
python -m venv myenv
# Windows
myenv\Scripts\Activate.ps1
# macOS/Linux
source myenv/bin/activate

step(3) Install dependencies:
pip install -r requirements.txt

step(4) Run the Streamlit app:
streamlit run frontend.py
